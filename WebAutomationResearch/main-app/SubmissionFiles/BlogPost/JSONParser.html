The Recursive Algorithm

The purpose of the recursive algorithm in this current state (7/3/21) is to render JSON String Value Pairs. Currently it doesn’t support other data types like dates, times, numbers, or lists. Here are the basic steps that it takes:-

Walkthrough

  1. Create an array, that will hold the HTML return components.
  2. Take an arbitrary JSON file and start the parse through it/search through it
  3. Each loop, it will check if it has hit a String Value Pair or an Array/Object
  4. If it’s a String Value Pair
    • It will declare a label stating the “String” value and an entry box. It will then append this to the array that was declared at the start
  5. If it’s an array or object
    • It will display the identifier of the array or object and then call the recursive function with the object/array.
    • This cycle will continue until it hit’s the end of the JSON. It will continue the append to the array the HTML components of each String Value
  6. Once it has hit the end of the of the JSON file. It will return the array to the React Renderer where it will be displayed.
