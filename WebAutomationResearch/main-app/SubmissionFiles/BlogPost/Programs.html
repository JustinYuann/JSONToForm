What Do You Need to Install Before Developing?

VSCode

This is where you’ll be writing your code for you frontend and backend. VSCode is a great text editor but not an IDE. Meaning you can’t compile the code directly using the program. However with the terminal you can easily compile Node.js.

Node.js

To install Node.js please follow this link. Ensure that you choose the right system, 32 or 64 bit. Once installed it will be a global path way inside your system, meaning you can run node commands in the terminal. If you want to know what tutorial I followed it's this video. If you have followed the instruction you know have your backend of the project!
