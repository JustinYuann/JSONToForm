Test your Node.js and Create your First React App

CMD: node-v

Open a Command Prompt if you're on windows and type in "node -v". If a something like "v14.15.0" appears, thats great. It means that your Node.js has been installed correctly on your system.

CMD: create-react-app

Open a Command Prompt in a folder where you want to create your first basic React app. Then type in the commands from the photo below! After a minute or so you should see a local host address and there will be your first React App!
