Traverse a Nested JSON in JavaScript

In this section, I’ll be walking through how to traverse a Nested JSON File. We won’t be focusing on the render of the data and we’ll just be printing out the value on the page.

Theoretical Steps

  • Set up a loop to go through the JSON. It will take in the JSON and iterate through String Value Pair
  • Now check if the value is either an Object, Array or just a primitive data type like numbers or strings
  • If it’s an Object or an Array, print the key of the String Value Pair and called the function with the String Value Pair
  • Else, print the String Value Pair and move onto the next pairing
  • The only time the Value of String Value Pair is printed is when the Value is identified as a String
