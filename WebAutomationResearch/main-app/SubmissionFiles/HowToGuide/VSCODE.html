Why Use VSCode Plugins

Above you can see the VSCode plugins I have. VSCode pluggins are useful to make coding easier by for exampling highlighting specific syntax or ensuring that there's matching brackets before compiling. Most of them have a description and are pretty self explanatory but I'll go through the main ones. To install this, please navigate to the extensions bar on the right hand side and use the search bar,

ESLint and Prettier

EsLint performs automated scans of your Javascript code which is the language Node.js and React use, to identify common syntax and style errors. Prettier scans your files for style issues and automatically reformats your code to ensure consistant rules are being followed for indentation, spacing, etc. These two do require a setting up so I suggest this video. I found these quite useful because it helps you learn the coding conventions of JavaScript but also makes your code "Prettier".
